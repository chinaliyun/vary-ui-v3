# Space 占位符

`Space`占位符, 主要用于快速添加一个指定宽度或者高度的空间,而不用再切换到 css 文件中去修改其他元素的样式

## 基础用法

<br/>

<Space-Base/>

@[code](../comps/Space/Base.vue)

## props

| name        | default | desc         |
| ----------- | ------- | ------------ |
| `w,width`   | ''      | 快速设置宽度 |
| `h,height` | ''      | 快速设置高度 |
